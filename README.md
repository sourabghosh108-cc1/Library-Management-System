# 📚 Library Management System

A beginner-friendly Python project that demonstrates the use of **lists**, **tuples**, **tuple packing and unpacking**, **indexing**, and **slicing** through a simple library management system.

This project was created as part of **CS 1101 – Programming Fundamentals** at the **University of the People**.

---

## Features

* Manage a list of library books
* Add new books using `append()`
* Remove damaged books using `remove()`
* Sort books alphabetically using `sort()`
* Store borrower information in immutable tuples
* Demonstrate tuple packing and unpacking
* Use indexing and slicing to manage weekly borrowing statistics
* Simple, well-commented code suitable for beginners

## Learning Objectives

This project demonstrates how different Python sequence data structures are used for different purposes:

* **Lists** store data that changes over time.
* **Tuples** store data that should remain unchanged.
* **Tuple unpacking** improves readability.
* **Indexing and slicing** provide efficient access to sequence elements.
